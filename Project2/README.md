# SIC/XE Machine2(Assemble 기능)
------
### 구현 명령(Shell)
-	type filename: filename에 해당하는 파일을 현재 디렉토리에서 읽어서 화면에 출력
-	assemble filename: filename에 햐당하는 소스파일을 읽어서 object파일과 리스팅 파일을 생성
-	symbol: 가자 최근의 assemble 과정에서 생성된 symbol table을 화면에 출력
